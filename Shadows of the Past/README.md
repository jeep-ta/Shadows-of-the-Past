# Shadows of the Past

A mysterious visual novel adventure game built with Pygame.

## Story
Awaken in a desolate world filled with ruins, decay, and monstrous creations. As you traverse this broken landscape, you must make choices that will determine your fate and the fate of the world. Will you find redemption, or are you doomed to repeat the cycle of destruction?

## Features
- **Interactive Storytelling**: Making choices that affect the game's ending (Karma system).
- **Atmospheric Audio**: Immersive background music.
- **Visuals**: Distinct background art for various scenes.
- **Typewriter Effect**: Engaging text presentation.

## Installation

1.  **Prerequisites**: You need Python installed on your system.
2.  **Dependencies**: Install Pygame.
    ```bash
    pip install pygame
    ```

## How to Play

1.  Run the game:
    ```bash
    python main.py
    ```
2.  **Controls**:
    - **Mouse**: Click to advance text and select choices.
    - **Resize**: The window is fully resizable.
